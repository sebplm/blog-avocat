# Meilleur Avocat Toulouse

Site éditorial local autour du dommage corporel, de l'indemnisation des victimes et des avocats spécialisés à Toulouse.

## Stack

- Next.js 14
- React 18
- Tailwind CSS
- Contenu éditorial généré depuis `lib/articles.ts`
- Pages piliers SEO générées depuis `lib/pillars.ts`

## Commandes

```bash
npm run dev
npm run build
npm run start
```

## SEO

Le site contient des pages piliers, un blog filtrable, un sitemap, un robots.txt, des métadonnées par page et du JSON-LD.
